- ✅ 生成提示词

- ✅ 复制结果

- ✅ 离线支持
- ✅ 像原生 App 一样安装

## 安装到 iPhone/iPad

### 第一步：获取网址

你需要先把网页上传到网上获取网址。

推荐使用 **Gitee Pages**（免费+国内速度快）：

1. 注册 Gitee：https://gitee.com
2. 创建新仓库，名为 `prompt-generator`
3. 上传本文件夹所有文件
4. 仓库设置 → Gitee Pages → 启用

获得网址后继续下一步。

### 第二步：Safari 打开

1. iPhone/iPad 打开 Safari
2. 输入你的网址
3. 打开网页

### 第三步：添加到主屏幕

1. 点击 Safari 底部的 **分享** 按钮 📤
2. 向下滚动，找到 **"添加到主屏幕"**
3. 点击 **添加**
4. 完成！

### 第四步：使用

1. 在主屏幕找到 **PromptAI** 图标
2. 点击打开
3. 第一次需要设置 API Key（点击右上角 ⚙️）

## API Key 获取

访问 https://platform.minimaxi.com/ 注册并获取 API Key
